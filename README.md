# 🐍 Snake Game con React
 
Un remake del clásico juego **Snake**, desarrollado con **React** y renderizado en **Canvas**.  
Este proyecto explora conceptos clave deprogramación de juegos y la integración de lógica 

---

## ✨ Características principales
- **Game loop con `setInterval`** para controlar la actualización del estado del juego.
- **Solución al bug de colisión con la cola**, asegurando que el juego detecte correctamente cuando la serpiente se muerde a sí misma.
- **Separación entre lógica y renderizado**, manteniendo el código modular y fácil de mantener.
- **Integración de Canvas con React**, aprovechando hooks y referencias para dibujar el juego en tiempo real.
